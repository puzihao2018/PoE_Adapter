## Requirements

* Support 802.3af, at, bt
* Maximum PoE input 71W
* Support 10M/100M/1G Ethernet, 2.5G or 10G better
* Support USB PD 60W(20V-3A) or higher, max to 65W
* Support USB PD 2.0 or higher, PPS support is better
